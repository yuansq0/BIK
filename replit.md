# Dynamic Visual Acuity Test

## Overview
A simple static HTML/JavaScript application for testing dynamic visual acuity. Users see numbers flash on screen and must remember and input them.

## Project Structure
- `index.html` - Main game interface with 5 difficulty levels
- `mmh.html` - Results comparison page showing user answers vs correct answers

## Technical Stack
- Pure HTML, CSS, and vanilla JavaScript
- No build tools or dependencies required
- Uses localStorage for passing data between pages

## Recent Changes
- **2024-11-25**: Initial import and Replit setup
  - Configured static file server workflow
  - Set up deployment configuration for static hosting

## Architecture
- Static single-page application
- No backend required
- Served via simple HTTP server on port 5000
